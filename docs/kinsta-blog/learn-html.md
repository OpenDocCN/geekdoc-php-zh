## 你应该从 HTML 中学到什么

总之，CSS 和 HTML 的知识可以让你成为一个熟练的开发人员。

Javascript 是另一种编程语言，你应该学习它来提高你的技能。它是一种编程语言，可以让你制作动态内容。您可以制作动画图像或让不同的多媒体元素在屏幕上飞舞。

Javascript 将使任何网站更吸引终端用户，并有可能增加他们在你的网站上的时间。
